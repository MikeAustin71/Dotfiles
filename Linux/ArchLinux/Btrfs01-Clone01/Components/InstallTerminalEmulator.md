# Install Alacritty

A fast, cross-platform, OpenGL terminal emulator.

## Features

- Scroll back through history with mouse
- Clear things with Ctrl+l (L)
- echo "https://ericmurphy.xyz" - You can click on it and go to website
- Ctrl+Shift+C = Copy
- Ctrl+Shift+V = Paste
